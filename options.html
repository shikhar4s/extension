Page Signal Capture

Maintains a resilient local WebSocket bridge to a Python GUI, captures a full-page PNG on demand, and returns the image for local saving and analysis.

Used as the direct fallback bridge target. Keep this as your local server unless you want a different default.

Fixed by the extension. On every reconnect it fetches this Pastebin raw URL, reads the ngrok target from there, and converts `http`, `https`, or `tcp` values into the correct WebSocket URL automatically.
